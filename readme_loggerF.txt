Readme for Logger F_Edited_Local

This is data on ambient temperature outside a Killdeer nest with 4 eggs in the parking lot of Star City BMX in Lincoln, Nebraska. The data was collected using an ibutton placed outside the nest on 06/23/2023 at 09:50 am. Readings were collected every 10 minutes. The ibutton was removed from the nest on the same day at 14:05 pm.

Variables:
Date/time: date and time of data point
Unit: Celsius
Value: Temperature of nest
